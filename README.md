# Smart ToDo App

A task management application that helps users organize daily activities efficiently.

## Features

- Add Tasks
- Mark Tasks as Completed
- Undo Completed Tasks
- Delete Tasks
- Clear All Tasks
- Local Storage Support
- Responsive User Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Purpose

This project was developed to improve JavaScript skills including DOM manipulation, event handling, and local storage management.

## Future Improvements

- Task Categories
- Due Dates
- Priority Levels
- Search Functionality
